Sombras y Transiciones del CSS

Sombras en CSS (box-shadow): se utiliza para crear una sombra detrás del contenedor, dándole profundidad visual y un efecto tridimensional.
Texto con sombra
Sombras en CSS (text-shadow): permite añadir una sombra al texto para hacerlo resaltar o dar un efecto de relieve.

Transiciones en CSS (transition, ease, duration): permiten cambiar suavemente el color del botón al pasar el cursor, haciendo la animación más fluida gracias al efecto ease y la duración de 0.4s.
